# Code of Conduct

## Our standard

We want TestSuits to be a welcoming, technically serious place. Everyone participating in this project — issues, PRs, discussions — is expected to behave professionally.

**Expected:**
- Focus on the technical substance. Critique ideas, not people.
- Assume good faith. Ask before accusing.
- Respect differing viewpoints and experience levels.
- Accept constructive feedback gracefully.

**Not welcome:**
- Personal attacks or harassment of any kind.
- Dismissive or demeaning language toward other contributors.
- Off-topic, disruptive, or low-effort content repeated after a warning.

## Enforcement

Project maintainers may remove, edit, or reject contributions that don't align with this code, and may ban (temporarily or permanently) contributors who repeatedly violate it.

## Reporting

If you see behavior that concerns you, email the maintainer at **salah.awad@outlook.com**. Reports will be handled confidentially. Do **not** open a public issue for conduct concerns.

## Scope

This code applies to all project spaces — the GitHub repository, its issues and PRs, and any project-related communication channels — and to public spaces when an individual is representing the project.
